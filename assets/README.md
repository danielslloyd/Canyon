# Assets Folder

Place your F22 3D model files in this folder.

## Supported Formats
- `f22.glb` or `f22.gltf` (GLTF/GLB format)
- `F22.glb` or `F22.gltf` (case variation)

## Model Requirements
- Low-poly F22 Raptor model
- GLTF/GLB format (recommended for web)
- Oriented with nose pointing in negative Z direction
- Reasonable scale (will be scaled 2x in game)

## Fallback
If no model is found, the game will use a simple procedural delta wing fighter.

## Where to Find Models
Free sources for F22 models:
- Sketchfab (with Creative Commons license)
- TurboSquid (free section)
- CGTrader (free models)
- Free3D

Make sure the model license allows use in your project!
